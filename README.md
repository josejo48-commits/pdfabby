# PDFabby 📄

Editor de PDF con OCR para celular y escritorio.

## Cómo usar
Visita: https://TU_USUARIO.github.io/pdfabby

## Funciones
- ✏️ Editar texto en PDFs escaneados (OCR)
- 🔤 Añadir texto nuevo
- 🖍️ Resaltar texto
- ◻ Borrar áreas
- 🖼️ Insertar imágenes
- ↩ Deshacer cambios
- ⬇ Descargar PDF editado
